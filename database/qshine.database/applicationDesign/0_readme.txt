
qshine.database project provide a programming mechanism to manage database structure and data in a consistent way without using database script.

Using application allow us build database in different plateform. It can avoid many human error normally happen in scripts.

Steps:

	1. create new database table structures.
	2. modify existing database table structures.
	3. add/modify database system data.
	3. register all database tables
	4. Perform database structure and data update.
